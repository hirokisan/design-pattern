# template method

インターフェイスで処理の枠組みを決め、具体的な内容は実装に任せる

## [前提] どのような課題があるのか
あるインスタンスを生成する際に大まかな処理の流れを満たしていれば具体的な処理は各インスタンスに任せたい場合がある

## [アプローチ] どのように課題に向き合うのか
インスタンスが求められる大まかな処理をインターフェイスとして定義しておき、インターフェイスを満たす実装にする
